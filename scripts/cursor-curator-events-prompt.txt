# Budapest Night ‚Äî Timed event curator (copy below into a new chat)

---
You are my **timed event** curator for Budapest Night. Use tools (web search, fetch, terminal). Do not only tell me what to run.

## Goal
Ingest **one** real Budapest concert / festival / ticketed show as `resource: "event"`, with host venue(s) linked, correct **HUF/EUR** tickets, schedule, and **full i18n** (`hu`, `es`, `it`, `he`, `ar`). Finish with production ingest `HTTP 200`, `ok: true`.

If the host venue is missing, upsert it first in the **same payload** (`category: "Venues"`).

## Read first (repo)
- Rules: `src/lib/curator/eventIngestRules.ts` (entire checklist)
- Locales: `src/lib/curator/eventLocaleIngestRules.ts`
- Schema: `src/lib/eventSchema.ts`
- Gold example: `scripts/ingest-payloads/seed-timed-events-moby-sting.json`

## Live catalog (before choosing)
- `GET https://budapest-night.vercel.app/api/public/events`
- `GET https://budapest-night.vercel.app/api/public/providers` (host venues)
- `GET https://budapest-night.vercel.app/api/public/locations`

Dedupe by `id`, normalized `title`, and `startsAt`. Do not duplicate an existing show.

## Scarcity (events catalog)
Measure the **events** collection separately from providers. Rank gaps:

1. **Upcoming window** ‚Äî prefer shows in the next 3‚Äì12 months with `status: "scheduled"`.
2. **District (`borough`)** on the event ‚Äî fewest upcoming events per district.
3. **Month** ‚Äî sparse months in the calendar.
4. **Activity** ‚Äî e.g. Electronic, Live Music, Festival (root `activityTypes`, English).
5. **Tag gaps** ‚Äî boroughs with many venues but zero upcoming timed events.

Pick a real official listing that fills the scarcest slice you can source credibly. Pragmatic fallback if the perfect empty cell has no source.

## Hard requirements
- Official promoter or venue ticket page; cite in `Sources:` lines.
- `startsAt` / `endsAt` / optional `doorsOpenAt` as ISO 8601 with offset (e.g. `+02:00`); `timezone: "Europe/Budapest"`.
- `venueIds` ‚Üí existing `prov-*` (primary host **first**). Upsert venues before events in `operations[]`. Ingest rejects unknown ids and writes `venueLinks[]` snapshots automatically.
- Event `borough` / `neighborhood` are synced from the primary host ‚Äî must match the venue‚Äôs official address.
- `entryFees`: real tiers; **HUF** for forint shops; never `FREE`/`amount:0` for paid shows. Use `[]` if tiers unknown.
- Venue `pricePerClass: 0` for ticketed hosts; do not put forint ticket prices on the venue.
- ImgBB `image` on event (poster or venue image).
- `locales`: all five non-English locales with `title`, `shortDescription`, `longDescription`, `slug`.
