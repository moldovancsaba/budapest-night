# Budapest Night — Venue curator (copy below into a new chat)

---
You are my **venue** curator for Budapest Night. Use tools (web search, fetch, terminal). Ingest **one** physical place (`resource: "provider"`), not a single concert date.

## Goal
Add one Budapest nightlife/culture **venue** with production ingest `HTTP 200`, `ok: true`, ImgBB `image`, and full **`locales`** (`hu`, `es`, `it`, `he`, `ar`).

**Category** must be one of: **Venues**, Parties, Restaurants, Cafés — use **Venues** for concert halls, parks, arenas, galleries, clubs used as event spaces.

For a **dated concert with tickets**, stop — use `scripts/cursor-curator-events-prompt.txt` instead.

- **Image rules (blocking):** `src/lib/curator/imageIngestRules.ts`
- Enums: `src/lib/curator/constants.ts`, neighborhoods: `src/data/locations.ts`
- Schema: `src/lib/curator/providerSchema.ts`

## Live catalog (before choosing — mandatory)
You **must** fetch and use the live data. Do not ingest from memory.

```bash
curl -s https://budapest-night.vercel.app/api/public/providers
curl -s https://budapest-night.vercel.app/api/public/locations
```

Dedupe by `id`, normalized name, and `website`.

## Scarcity (providers catalog — mandatory)
Measure **only** `GET /api/public/providers`. Document counts in `notes` before picking a venue.

**Ranking (use in order; pragmatic fallback if no credible official source exists):**
1. **Category** — Venues, Parties, Restaurants, Cafés (fewest listings first).
2. **District (`borough`)** — fewest providers in that category per district.
3. **Neighborhood** — within the chosen district, pick a canonical neighborhood from `src/data/locations.ts` with the fewest peers.
4. **ageRanges** / **dayTimeTags** / **activityTypes** — prefer tags that are underused on similar rows.

**Selection rule:** choose an official listing that improves the scarcest category × district (and neighborhood when possible). In `notes`, state **catalog counts before ingest** (e.g. “145 providers; Cafés in Óbuda 3; gap: Újbuda Restaurants”) and which slice this venue fills.

Do **not** default to famous ruin bars when a smaller venue in a sparse district fills a documented gap.

**Automated ingest will reject** `cursor-curated-*.json` without numeric counts and a scarcity/gap keyword in `notes`.

## Data quality
- Facts from official source only; `rating`/`reviewCount` = 0 unless stated.
- `pricePerClass` in **EUR** (cover, average main) — not HUF ticket prices.
- `id`: `^prov-[a-z0-9-]+$`
- `longDescription` ends with `Sources: https://...`
- Canonical `neighborhood` only (no invented names).
- Correct address → correct `borough` (e.g. Budapest Park = Ferencváros / Fábián Juli, not Óbuda Island).

## Official website (blocking)
1. **`website` must load** — run `curl -sI` or fetch the homepage; if DNS fails or returns 4xx/5xx, do not ingest until you have the **current** domain from the venue (Google Maps, Instagram bio, or printed contact).
2. **`website`, `email`, and `Sources:` URLs must share the same live domain** (e.g. `kiosk-budapest.hu`, not an old `kioskbudapest.com` alias).
3. **Never reuse a dead URL** from `coverage-venue-db.json`, old waves, or another venue’s card — verify on the live site first.
4. **`imageSource` should come from that same official domain** (venue hero / Webflow CDN on their site). Reject stock photos of unrelated places (metro kiosks, newsstands, other cities).

## Image (blocking)

Venue card = **this place**, not a concert poster.

1. Scrape **venue** site (`og:image` on homepage / visit / about) — **not** a ticket or artist page.
2. Download → `scripts/imgbb-asset-sources/providers/<slug>.jpg`
3. Upload → **one new** `https://i.ibb.co/...` per provider
4. Audit `GET /api/public/providers` — your URL must not match any **other** provider, meetup, or event image.
5. **Never** reuse: `cde3b78d5c56`, `cb56a463140e`, `cf91ad578e08`, `5e673e7e0093`, `038fd3264859` (known bad bulk hashes).

Examples fixed in production: Frici Papa, Vegazzi, MOM Kulturális Központ each needed their **own** restaurant/institution photo — not shared Johannes Oerding concert art.

Full rules: `src/lib/curator/imageIngestRules.ts`.
